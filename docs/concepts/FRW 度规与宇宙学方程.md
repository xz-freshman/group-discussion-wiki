# FRW 度规与宇宙学方程

## FRW 度规（Friedmann–Roberson–Walker Metric）

### 基本假设

宇宙是**均匀各向同性**的（Homogeneous & Isotropic）：
- **均匀性**：宇观尺度上，任意位置的物理规律相同
- **各向同性**：以固定参考系测量，所有方向相同

### 度规形式

```
ds² = -c²dt² + a(t)² · (dr² + r²dΩ²)
```

其中：
- **a(t)**：尺度因子，唯一自由函数
- **r**：共动坐标（不随宇宙膨胀而变化）
- **dΩ² = dθ² + sin²θ dφ²**：球面度规

### 为什么 dt² 前系数必须是 c²？

通过定义新的时间坐标，总可以把 dt² 前系数归一化为常数 c²（光速）。如果不是 c²，可以被吸收进 a(t) 的定义中。因此 FRW 度规中只有**一个**自由函数 a(t)。

### 参数 K（曲率）

| K 值 | 宇宙类型 |
|------|---------|
| K > 0 | 封闭宇宙（正曲率） |
| K = 0 | **平坦宇宙**（当前观测结果） |
| K < 0 | 开放宇宙（负曲率） |

当前观测 K ≈ 0，暴涨理论（Inflation）可以自然解释。

## FRW 方程（Friedmann Equations）

从 FRW 度规出发，结合爱因斯坦场方程 R_μν - (1/2)g_μνR = (8πG/c⁴) T_μν：

### 第一弗里德曼方程
```
H(t)² = (ȧ/a)² = (8πG/3) ρ - k c²/a² + Λc²/3
```
右边三项：物质/辐射的引力效应（吸引）｜曲率项｜暗能量的排斥效应

### 第二弗里德曼方程（加速度方程）
```
ȧ/a = -(4πG/3)(ρ + 3p/c²) + Λc²/3
```
这一方程直接给出宇宙是否在加速膨胀：取决于 (ρ + 3p/c²) 是否足够小。

> 📎 **补充知识①：为什么第二方程中出现了压强 p？**
>
> 在牛顿极限下，ȧ/a = -(4πG/3)ρ 是大家熟悉的。但到了 GR，p 出现在方程中是因为引力不仅来源于能量密度 ρ，也来源于压强 p（动量流密度的散度）。这就是为什么辐射（p = ρ/3）也产生引力，而且它的引力效应比同等质量的物质更强（1/3 额外的项来自 3p）。
>
> 对于暗能量，p = -ρ < 0，其负压产生的引力是排斥的（加速膨胀）。

## 动力学方程的物理意义

### 能量守恒方程
从协变守恒 ∇_μ T^{μν} = 0 可以推导出能量守恒方程：
```
dρ/dt = -3(ρ + p/c²) · (ȧ/a)
```

> 📎 **补充知识②：三个组分的稀释公式推导**（讲座未展开）
>
> 能量守恒方程对各组分分别成立：
>
> - **辐射**（p = ρ/3）：dρ/ρ = -4 da/a ⟹ ρ ∝ a⁻⁴
> - **物质**（p ≈ 0，冷暗物质）：dρ/ρ = -3 da/a ⟹ ρ ∝ a⁻³
> - **暗能量**（p = -ρ）：dρ/dt = 0 ⟹ ρ = 常数
>
> 这正是讲座中提到的三种稀释方式的来源。数学上干净地来自能量守恒方程，不需要逐个记忆。

### 曲率项的命运
第一方程中 k c²/a² 项随 a 增大而迅速衰减（∝ a⁻²）。这意味着即使宇宙初始有显著的曲率，只要膨胀足够久，曲线项最终也会变得可以忽略，宇宙趋向平坦。

> 📎 **补充知识③：平坦性问题的定量表述**
>
> 从第一弗里德曼方程可以导出：
> ```
> |K| = a² |H₀²(Ω_tot - 1)|
> ```
>
> 今天 Ω_tot ≈ 1（|Ω_tot - 1| < 0.001），而宇宙年龄约 10¹⁰ 年。回推到普朗克时代（t ≈ 10⁻⁴³ 秒），a 约小 10⁶⁰ 倍，此时要求 |Ω - 1| 的初始值精确到 10⁻⁶⁰。这不是微调，而是需要解释的"精细调节"问题。
>
> **暴涨的贡献**：暴涨把 a 再放大约 e^(60) ≈ 10²⁶ 倍（标准暴涨约 60 个 e-fold），相当于把回推深度降低 26 个数量级，从而把 10⁻⁶⁰ 问题变成 10⁻³⁴——这在现代物理中完全可接受。

## 状态方程（Equation of State）

各组分可以用**状态方程参数 w**（p = w·ρ）来统一描述：

| 组分 | w | 备注 |
|------|---|------|
| 辐射 | 1/3 | 极端相对论气体 |
| 物质 | 0 | 非相对论冷物质 |
| 暗能量 | -1 | 宇宙学常数（w = -1 是临界值）|
| 暴涨场 | ≈ -1 | 暴涨期间 a ∝ e^(Ht)，H 近似常数 |

> 📎 **补充知识④：w ≠ -1 的暗能量**
>
> 如果暗能量不是严格常数（w ≠ -1），叫做"动态暗能量"（Quintessence）。此时 Ω_Λ 会随时间变化，可以导致比宇宙学常数更丰富 的宇宙学前景。
>
> Hubble Tension 的一种可能解释正是：真实的暗能量 w 随时间变化，而非标准模型中的固定 -1。

## 与广义相对论的关系

- FRW 度规是 GR 在宇宙学情景下的最简解
- 描述的是最大尺度（> ~Gpc）的宇宙整体结构
- 在小尺度上（星系团以下）不适用（不均匀性显著）

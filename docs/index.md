=== "💻 计算机网络"
    **主题一：互联网是如何层层叠加的**

    从 MAC 地址出发，经过集线器、交换机、路由器，到 IP、DNS、HTTP/HTTPS。每一层都是在前一层基础上"叠加"，而不是"替换"。

    [计算机网络基础](concepts/计算机网络基础.md){ .md-button }

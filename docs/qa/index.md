# Q&A

> 把讨论里真正有价值的问题单独拎出来，方便快速查和快速测。

---

## 这个分区适合谁

- 你已经看过摘要，想快速复盘
- 你只想回答某个具体问题
- 你想用“问题驱动”的方式复习

---

## 使用建议

!!! tip "最省时间的用法"
    先自己回答一遍问题，再展开看答案。这样比直接看正文更容易发现自己到底哪没理解。
